# GST-CMDB

Система учёта IT-активов, лицензий и уведомлений (CMDB).

## Требования
- PHP 8.0+
- MySQL 8.0+
- Обычный виртуальный/корпоративный хостинг с поддержкой PHP, MySQL

## Установка

1. **Создать базу данных (например, через phpMyAdmin):**
   - Имя БД: `cmdbbd`
   - Пользователь: `cmdbuser`
   - Пароль: `cmdbpass123`

2. **Импортировать структуру БД:**
   - В phpMyAdmin или через консоль MySQL выполнить файл `db_schema.sql`

3. **Загрузить все файлы проекта в корень сайта** (обычно папка `public_html` или `www`)

4. **Проверьте, что в файле `config.php` настройки подключения к базе совпадают с вашими**

5. **Откройте сайт в браузере** (`index.php`). Должна отобразиться стартовая страница GST-CMDB.

6. **Cron для уведомлений**: укажите ежедневный запуск скрипта (будет предоставлен `cron_notify.php`).

## Дополнительно
- Управление системой, настройка email, ролями, журналирование и т.д. реализуются из административного интерфейса.
